# database / drizzle commands

## Available Commands

```bash
# Setup environment (copies .env.example to .env)
pnpm db:setup

# Check configuration
pnpm db:check

# Generate migrations
pnpm db:generate

# Apply migrations to D1
pnpm db:push

# Open Drizzle Studio
pnpm db:studio

# Drop database (use with caution)
pnpm db:drop
```
